## company接口
请求url   http://47.98.36.72:10000/search/company_detail
请求消息体
```python
  {  
    "area":"江苏",
    "page":"1",
    "per_page":"10000",
    "startn":"100"
}
```
area是省份，page页面号，per_page每页展示数据量，startn数据返回时的起始偏移量

```python
{
```
